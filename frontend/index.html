// 语义熵知识边界感知

基于 Kuhn et al., Nature 2024（引用 1271 次）
核心思想：对同一输入多次采样 LLM 回答，计算语义熵。
高熵 → 模型不确定 → 输入可能超出知识边界（未知威胁）
低熵 → 模型确定 → 输入在知识边界内（已知模式）
需要 LLM_ENABLED=true 且 SEMANTIC_ENTROPY_ENABLED=true
计算语义熵中（需要多次采样，约 10-30s）
